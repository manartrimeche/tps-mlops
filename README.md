# README

Ce dossier contient plusieurs projets MLOps et IA. Voici un aperçu de chaque projet :

## iris-ai-service

Service API pour la classification d'iris avec FastAPI, Docker, monitoring (Prometheus, Grafana).
**Accès :** Ouvrir le dossier `iris-ai-service`.
**Démarrage rapide :** Voir le README du dossier pour les instructions.

## mlflow-cv-yolo

Expérimentation MLflow pour YOLOv8 sur COCO128, gestion des runs, scripts d'entraînement, DVC pour les données.
**Accès :** Ouvrir le dossier `mlflow-cv-yolo`.
**Démarrage rapide :** Voir le README du dossier pour les instructions.

## mlops-dvc-iris

Pipeline MLOps pour iris avec DVC, gestion des données et des images, orchestration via Docker Compose.
**Accès :** Ouvrir le dossier `mlops-dvc-iris`.
**Démarrage rapide :** Voir le README du dossier pour les instructions.

## optuna-cv-yolo

Optimisation d'hyperparamètres YOLOv8 avec Optuna, DVC, MLflow, scripts d'entraînement et pipelines ZenML.
**Accès :** Ouvrir le dossier `optuna-cv-yolo`.
**Démarrage rapide :** Voir le README du dossier pour les instructions.

## zenml-cv-yolo

Pipeline MLOps complet avec ZenML pour YOLOv8, DVC, MLflow, scripts d'entraînement et optimisation.
**Accès :** Ouvrir le dossier `zenml-cv-yolo`.
**Démarrage rapide :** Voir le README du dossier pour les instructions.

## deploy-cv-yolo

Déploiement d'un modèle YOLO avec TorchServe, API Gateway, scripts de packaging et CI/CD GitLab.
**Accès :** Ouvrir le dossier `deploy-cv-yolo`.
**Démarrage rapide :** Voir le README du dossier pour les instructions sur le déploiement, la configuration et les tests.

---

Chaque dossier contient un README spécifique, des scripts, des configurations et des exemples d'utilisation.
