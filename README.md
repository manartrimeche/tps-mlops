# README

Ce dossier contient plusieurs projets MLOps et IA. Voici un aperçu de chaque projet :

## iris-ai-service
Service API pour la classification d'iris avec FastAPI, Docker, monitoring (Prometheus, Grafana).

## mlflow-cv-yolo
Expérimentation MLflow pour YOLOv8 sur COCO128, gestion des runs, scripts d'entraînement, DVC pour les données.

## mlops-dvc-iris
Pipeline MLOps pour iris avec DVC, gestion des données et des images, orchestration via Docker Compose.

## optuna-cv-yolo
Optimisation d'hyperparamètres YOLOv8 avec Optuna, DVC, MLflow, scripts d'entraînement et pipelines ZenML.

## zenml-cv-yolo
Pipeline MLOps complet avec ZenML pour YOLOv8, DVC, MLflow, scripts d'entraînement et optimisation.

---

Chaque dossier contient un README spécifique, des scripts, des configurations et des exemples d'utilisation. Pour accéder à un projet, ouvrez le dossier correspondant.
